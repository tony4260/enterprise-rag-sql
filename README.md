# Enterprise RAG SQL

Enterprise-oriented Retrieval-Augmented Generation (RAG) prototype built with:

- Python
- SQLite
- SQLAlchemy
- ChromaDB
- sentence-transformers

## Features

- Structured metadata storage
- Metadata-aware retrieval
- Permission-aware search
- Governance filtering
- Semantic vector search
- Similarity thresholds
- Hybrid retrieval architecture

## Architecture

User Query
→ SQL Metadata Filter
→ Embedding Model
→ ChromaDB Vector Search
→ Permission Validation
→ Safe Retrieval Response

## Key Concepts

- SQL as source of truth
- Vector database as semantic index
- Filter first, retrieve second
- Governance-aware AI systems
- Hybrid structured + semantic retrieval
